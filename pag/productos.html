Mantenimiento de Equipos

Mantenimiento Completo

El Mantenimiento completo es el mantenimiento que se le hace a un equipo 1 o 2 vez al año para que este siempre este actualizado y libre de datos inesesarios en el equipo, en el proseso se formatea el equipo, se instala el sistema operativo y todas las aplicaciones nesesarias para su correcto funcionamiento, esto dependera del rendimiento del equipo, ya que el objetivo de este mantenimiento es mejorar el rendimiento del equipo.

Mantenimiento Preventivo

Este mantenimiento se hace cada 2 o 3 meses, esto con el fin de prevenir la presencia de malwares y exeso de archivos basura almacenados, que reducen la prodctividad del equipo, este mantenimiento se puede hacer en ficico o atraves de la aplicacion TeamViewer, en este proseso se elimina archivos tamporales y historial de navegacion inecesaria.
