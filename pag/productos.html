Mantenimiento

Mantenimiento Completo

El Mantenimiento completo es el mantenimiento que se le hace a un equipo 1 o 2 veces al año, para que este siempre este actualizado y libre de datos innecesarios en el equipo, en el proceso se formatea el equipo, se instala el sistema operativo y todas las aplicaciones necesarias para su correcto funcionamiento, esto dependerá del rendimiento del equipo, debido a que el objetivo de este mantenimiento es mejorar el rendimiento de la maquina desde SOFTWARE.

Mantenimiento Preventivo

Este mantenimiento se hace cada 2 o 3 meses, con el fin de prevenir la presencia de malwares y exceso de archivos innecesarios almacenados, que reducen la productividad del equipo, este mantenimiento se puede hacer en físico o atreves de la aplicación TeamViewer, en este proceso se elimina archivos temporales e historial de navegación innecesaria.


Detalles

Re-potencialización

La re-potencialización de un equipo es el proceso de analizar la capacidad y los componentes de un equipo, para mejorarlos y re-potencializarlos, con el fin de mejorar el equipo y su rendimiento atreves de la memoria RAM, Tipo de Almacenamiento, y versión de Windows.
